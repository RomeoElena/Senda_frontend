# Senda — Frontend

Frontend de la aplicación Senda, una plataforma de gestión de armario personal sostenible.

## Stack

- Next.js 16 (React)
- Tailwind CSS
- JavaScript

## Arrancar en local

npm install
npm run dev

## Variables de entorno

Crea un archivo .env.local con:
NEXT_PUBLIC_API_URL=http://localhost:5000/api
